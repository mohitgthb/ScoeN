# ScoeN - Student Community of Engineering Notes

## 📌 About the Project
SCOEN (Student Community of Engineering Notes) is a platform that enables students to share and access notes seamlessly. It aims to bridge the gap between students by providing a collaborative space for exchanging study materials, lecture notes, and academic resources.

## 🚀 Features
- 🔍 **Search & Discover**: Find notes based on subjects, topics, or keywords.
- 👥 **Community-Driven**: A collaborative platform where students help each other.
- 📖 **Categorized Content**: Notes are organized by subject and semester.

## 🔧 Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: OAuth
- **Storage**: MongoDB clusters and Google Cloud for optimized storage

## 📜 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/scoen.git
   cd scoen
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables:
   ```plaintext
   MONGO_URI=your_mongodb_connection_string
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## 🛠️ Usage
- Sign up or log in to your account.
- Upload notes and categorize them by subject.
- Browse, and access useful notes.
